# Calculator Library Demo

A simple Java calculator library built as a reusable JAR and imported into another Java project.

This project demonstrates:

- Creating a reusable Java library
- Packaging the library as a `.jar`
- Using Maven to build the project
- Writing unit tests with JUnit
- Importing and using the library in another project

---

# Features

The calculator library supports:

- Addition
- Subtraction
- Multiplication
- Division
- Modulus

---

# Technologies Used

- Java
- Maven
- JUnit 5

---

# Exception Handling

The library throws `IllegalArgumentException` for invalid mathematical operations such as:

- Division by zero
- Modulus by zero
